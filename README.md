# SurfSense Issues

This is the **public issue tracker** for [SurfSense](https://github.com/Aimino-Tech/SurfSense) — an AI-powered knowledge management platform with unlimited sources, 25+ connectors, real-time collaboration, and agentic AI.

## Purpose

- **Report bugs** — Something not working? Let us know with a detailed bug report
- **Request features** — Have an idea? Describe what you need
- **Report writing quality issues** — If you're using the Writing Agent, report scoring inaccuracies or quality regressions
- **Track our progress** — See what we're working on and what's planned

## Getting Help

For usage questions, documentation, and guides, visit the [main SurfSense repository](https://github.com/Aimino-Tech/SurfSense).

For SLA expectations and support channels, see [SUPPORT.md](./SUPPORT.md).

## Reporting Security Vulnerabilities

If you discover a security vulnerability, please follow the disclosure process in [SECURITY.md](./SECURITY.md). **Do not** file a public issue for security vulnerabilities.
